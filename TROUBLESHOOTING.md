## 로그인 정보

Mock API를 사용하므로, 다음 계정으로 로그인하세요:

```
이메일: test@example.com
비밀번호: password123
```

## 대시보드가 안 보일 때

1. **로그인이 되어 있는지 확인**
   - 위의 테스트 계정으로 로그인하세요
   - 로그인하지 않으면 데이터가 표시되지 않습니다

2. **로컬 스토리지 초기화**
   - 브라우저 개발자 도구(F12) → Application → Local Storage
   - `horizon_swagger_` 로 시작하는 모든 항목 삭제
   - 페이지 새로고침

## 학교 연결이 안 될 때

**학교 코드**: `BSSM`

학교 연결 시 다음 단계를 따르세요:

1. 학교 연결 페이지 이동
2. "연결하기" 버튼 클릭
3. 프롬프트에 `BSSM` 입력
4. 연결 완료

만약 실패한다면:
1. 로그인 상태 확인
2. 로컬 스토리지 초기화
3. 다시 시도

## 팀 생성 문제

팀 생성 후 초대 코드가 표시됩니다:
- 서버에서 생성된 초대 코드를 사용합니다
- 기본 코드 예시: `STUDY2024`
